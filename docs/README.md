### 快速开始

- `git clone https://github.com/xieyezi/es6-vuepress-version.git`
- `npm run docs:dev`

### 一些命令

本地预览:

```
npm run docs:dev
```

打包:

```
npm run docs:build
```

部署到 github pages:

```
npm run deploy
```
